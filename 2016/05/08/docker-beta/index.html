收到 Docker Beta 的通知信了！

2016-05-08
docker

前兩天終於收到 Docker Beta for Mac 的通知信了，這次的 Docker Beta 在 Mac 和 Windows 底下擺脫了 VirtualBox 的依賴，並且在 Unikernel 技術的加持下，未來每個 container 只會留下需要的部份，打包成最小化的獨立作業系統。

以後找個時間試試看用 RancherOS 或 Weave 來組織 MicroService 好了。
