Koa.js 踩坑筆記

以 Koa.js 2.0.0-alpha.3 版本為主。

ctx.body 沒反應

當 ctx.body 在 Promise 的函式裡時（如 co），ctx.body 不會對頁面有任何反應，需要在該 function 前加 return。

自訂 middleware 的時候也是如此

1.0 與 2.0 版本問題

Koa.js 的大部分 middleware 都已經開始支援 2.0，鑑於是 alpha 的關係，要安裝這些元件時要加 @next 的 tag，尚未支援的在使用時可以安裝 koa-convert 來轉換。
